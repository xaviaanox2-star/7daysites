# 7DaySites Landing Page

Welcome to the 7DaySites landing page project!

## Project Overview
This project provides a customizable landing page for promoting services, providing contact information, and integrating bookings via WhatsApp.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/xaviaanox2-star/7daysites.git
   cd 7daysites
   ```
2. **Install Dependencies**:
   If you have any dependencies, install them using:
   ```bash
   npm install
   ```
3. **Run the Project Locally**:
   To preview the changes, run:
   ```bash
   npm start
   ```

## Customizing WhatsApp Button
To customize the WhatsApp button:
1. Open the `index.html` file.
2. Locate the WhatsApp button section.
3. Change the phone number in the URL to the desired contact number.
   ```html
   <a href="https://wa.me/1234567890">Chat with us on WhatsApp</a>
   ```

## Booking Links
To set up booking links:
1. Navigate to the `services.html` file.
2. Update the anchor tags with your booking URL.
   ```html
   <a href="https://yourbookinglink.com">Book Now</a>
   ```

## Contact Form Integration
For integrating the contact form:
1. Use a service like Formspree or Google Forms.
2. Update the action URL in the contact form to the provided service link.
   ```html
   <form action="https://formspree.io/yourform" method="POST">
   ```

## GitHub Pages Deployment
To deploy using GitHub Pages:
1. Push the changes to the `main` branch.
2. Navigate to your repository settings.
3. Under "GitHub Pages", select the branch you want to deploy from (typically `main`).

Your site will be live at `https://xaviaanox2-star.github.io/7daysites/` within a few minutes after changes.
